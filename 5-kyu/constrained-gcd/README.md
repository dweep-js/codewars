<h2><a href=https://www.codewars.com/kata/59f61aada01431e8c200008d/train/go target="_blank">Constrained GCD</a></h2><h3>5 kyu</h3><p>In this Kata, you will be given two numbers, n and k and your task will be to return the k-digit array that sums to n and has the maximum possible GCD.</p><p>For example, given <code>n = 12, k = 3</code>, there are a number of possible <code>3-digit</code> arrays that sum to <code>12</code>, such as <code>[1,2,9], [2,3,7], [2,4,6], ...</code> and so on. Of all the possibilities, the one with the highest GCD is <code>[2,4,6]</code>. Therefore, <code>solve(12,3) = [2,4,6]</code>.</p><p>Note also that digits cannot be repeated within the sub-array, so <code>[1,1,10]</code> is not a possibility. Lastly, if there is no such array, return an empty array.</p><p>More examples in the test cases.</p><p>Good luck!</p>
